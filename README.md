# SIDAS - Smart IT Desk Automation System

A web-based complaint and issue management system for colleges, offices, and institutions.

## Features
- User Complaint Registration
- Admin Dashboard
- Status Tracking
- Department Wise Management
- Reports Generation
- Secure Login

## Technologies Used
- Java
- Spring Boot
- MySQL
- HTML/CSS/Bootstrap
- JavaScript

## How to Run
1. Import in IntelliJ / Eclipse
2. Configure MySQL database
3. Run SidasApplication.java
4. Open localhost:8080
